## 题意

给出一个正整数 $N$，从大到小输出所有 $\leq N$ 的非负整数。

## 数据范围

- $1 \leq N \leq 100$

- $N$ 为整数。

## 输入格式

第一行输入一个正整数 $N$。

## 输出格式

输出若干行，每行为第 $i$ 大的 $\leq N$ 的非负整数。

## 样例

### 输入样例 #1

```
3
```

### 输出样例 #1

```
3
2
1
0
```

### 输入样例 #2

```
22
```

### 输出样例 #2

```
22
21
20
19
18
17
16
15
14
13
12
11
10
9
8
7
6
5
4

```
