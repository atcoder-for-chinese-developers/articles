## 题意

给出一个长度为 $N$ 的序列 $A=(A_1,A_2,\dots,A_N)$。

现要求找出一个长度为 $M$ 的子序列 $B=(B_1,B_2,\dots,B_M)$，使得 $\displaystyle \sum_{i=1}^{M} i \times B_i$ 的值最大。

输出这个最大的权值。

## 限制

- $1≤M≤N≤2×10^5$
- $- 2 \times 10^5 \le A_i \le 2 \times 10^5$
- 所有读入的数值都是整数。
