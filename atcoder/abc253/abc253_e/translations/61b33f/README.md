## 题意
有多少个长度为 $N$ 的正整数序列 $A=(A_1,\dots,A_N)$，每个元素的值在 $1$ 到 $M$ 之间，且满足相邻两个数的差 $\ge K$？

由于数量可能会很大，请将答案对 $998244353$ 取模。

## 数据范围

- $2 \le N \le 1000$

- $1 \le M \le 5000$
- $0 \le K \le M-1$
- 所有读入的数值都是整数。

## 输入格式

输入是由标准输入提供的，格式如下：

```
N M K
```

## 输出格式

输出这个数量，对 $998244353$ 取模。

## 样例

### 样例一

```
2 3 1
```

```
6
```

有这样 $6$ 个序列满足要求：

- $(1,2)$
- $(1,3)$
- $(2,1)$
- $(2,3)$
- $(3,1)$
- $(3,2)$

### 样例二

```
3 3 2
```

```
2
```

有这样 $2$ 个序列满足要求：

- $(1,3,1)$
- $(3,1,3)$

### 样例三

```
100 1000 500
```

```
657064711
```

注意要将答案对 $998244353$ 取模。
