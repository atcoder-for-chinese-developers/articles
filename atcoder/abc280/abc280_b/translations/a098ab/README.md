## 题意

给出一个长度为 $N$ 的序列 $S$，请构造出一个长度为 $N$ 的序列 $A$，使其满足如下条件：

- $\forall k \in [1,N], \sum_{i=1}^{k}{A_i}=S_k$

题目有且仅有唯一解。

## 数据范围

- $1 \leq N \leq 10$

- $-10^9 \leq S_i \leq 10^9$

- 所有输入均为整数。

## 输入格式

第一行输入一个正整数 $N$。

第二行输入 $N$ 个整数 $S_i$。

## 输出格式

第一行输出 $N$ 个整数 $A_i$。

## 样例 

### 输入样例 #1

```
3
3 4 8
```

### 输出样例 #1

```
3 1 4
```

### 样例解释 #1

- $\sum_{i=1}^{1}{A_i}=3=S_1$

- $\sum_{i=1}^{2}{A_i}=4=S_2$

- $\sum_{i=1}^{3}{A_i}=8=S_3$
