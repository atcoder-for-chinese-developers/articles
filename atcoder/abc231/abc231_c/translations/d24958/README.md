## 题意

有 $N$ 个学生，第 $i$ 个学生的身高为 $A_i$。  
$Q$ 个询问，第 $j$ 个询问身高大于等于 $x_j$ 的学生数量。

## 数据范围

- $1 \le N,Q \le 2 \times 10^5$
- $1 \le A_i \le 10^9$
- $1 \le x_j \le 10^9$
- 输入都是整数。 

## 输入格式

第一行两个正整数 $N,Q$ 表示学生人数和询问数。  
接下来一行 $N$ 个正整数 $A_i$ 表示每个学生的身高。  
接下来 $Q$ 行每行一个正整数 $x_j$ 表示询问。

## 输出格式

输出 $Q$ 行一行一个整数表示答案。

## 样例

### 样例输入1

```plain
3 1
100 160 130
120

```

### 样例输出1

```plain
2

```

### 样例解释1

第 $2$ 和第 $3$ 个学生的身高大于等于 $120$。
