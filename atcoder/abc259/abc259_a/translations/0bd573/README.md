## 题意

op dd_d 先生 $n$ 岁了，现在他的高度为 $T$。

在 $x$ 岁前他身高每年增加 $D$。在 $x$ 岁后他的身高不会变化。  

请输出 op 先生 $m$ 岁生日的身高。  

## 数据范围

$0 \leq m < n \leq 100,1 \leq x \leq n,1 \leq T \leq 200,1 \leq D \leq 100$，保证所有数都是整数。  

## 输入格式

一行五个整数 $n,m,x,T,D$。

## 输出格式

一行一个整数表示 op 先生的身高。  

## 样例

### 样例输入 1

```
38 20 17 168 3
```

### 样例输出 1

```
168
```

### 样例解释

op 先生在他 $38$ 岁时身高为 $168$，此外，$17$ 岁到 $38$ 岁之间身高没有变化。所以他的身高为 $168$

### 样例输入 2

```
1 0 1 3 2
```

### 样例输出 2

```
1
```

### 样例输入 3

```
100 10 100 180 1
```

### 样例输出 3

```
90
```
