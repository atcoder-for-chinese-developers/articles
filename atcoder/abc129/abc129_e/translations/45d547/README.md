## 题意

给定二进制正整数 $L$。求有多少对非负整数 $(a,b)$ 满足下列条件：
- $a+b\le L$
- $a+b=a\;\mathbf{xor}\;b$

由于存在很多这样的非负整数对，请输出答案对 $10^9+7$ 取模的结果。

$\mathbf{xor}$ 是二进制按位异或。

## 数据范围

- $L$ 以无前导零的二进制形式给出。
- $1\le L< 2^{10001}$

## 输入格式

一行一个二进制整数 $L$。

## 输出格式

输出一个整数，答案对 $10^9+7$ 取模的结果。

## 样例输入 #1
```
10
```

## 样例输出 #1
```
5
```

## 样例解释1

$5$ 对分别是：$(0,0),(0,1),(1,0),(0,2),(2,0)$。

## 样例输入 #2

```
1111111111111111111
```

## 样例输出 #2

```
162261460
```
