## 题意  

给定一个长度为 $N$ 的序列 $A$。         
输出满足 $1\le i < j \le N$ 且 $A_i \neq A_j$ 的数对 $(i,j)$ 数量。 

## 数据范围

$1\le N\le 3\times 10^5$，                     
$1\le A_i\le 10^9$，                       
所有输入的数都是整数。  

## 输入格式

第一行一个整数 $N$。        
第二行 $N$ 个整数 $A_i$。       
          
## 输出格式

合法的 $(i,j)$ 对数。

## 样例

### 输入1 :
```
3
1 7 1
```

### 输出1 :
```
2
```

### 输入2 :
```
10
1 10 100 1000 10000 100000 1000000 10000000 100000000 1000000000
```

### 输出2 :
```
45
```

### 输入3 :
```
20
7 8 1 1 4 9 9 6 8 2 4 1 1 9 5 5 5 3 6 4
```

### 输出3 :
```
173
```

## 样例解释

对于样例一，合法的数对有 $(1,2),(2,3)$。
