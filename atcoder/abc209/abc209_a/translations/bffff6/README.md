## 题意  

问有多少个整数 $i$ 满足 $A\le i\le B$。 

## 数据范围

$1\le A,B\le 100$，         
$A,B$ 是整数。               

## 输入格式

两个整数 $A,B$。
          
## 输出格式

'Yes' 或 `No`。   

## 样例

### 输入1 :
```
2 4
```

### 输出1 :
```
3
```

### 输入2 :
```
10 100
```

### 输出2 :
```
91
```

### 输入3 :
```
3 2
```

### 输出3 :
```
0
```

## 样例解释

对于样例 1，合法的 $i$ 有 $2,3,4$。
