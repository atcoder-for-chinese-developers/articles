## 题意  

问投掷一个写着 $1$ 到 $6$ 的骰子 $A$ 次，正面朝上的点数和是否可能是 $B$。   

## 数据范围

$1\le A\le 100$，    
$1\le B\le 1000$，     
$A,B$ 是整数。               

## 输入格式

两个整数 $A,B$。
          
## 输出格式

'Yes' 或 `No`。   

## 样例

### 输入1 :
```
2 11
```

### 输出1 :
```
Yes
```

### 输入2 :
```
2 13
```

### 输出2 :
```
No
```

### 输入3 :
```
100 600
```

### 输出3 :
```
Yes
```

## 样例解释

对于样例 1，一种可能的情况是 $5+6=11$。
