## 题意 

对于一棵无向树 $t$，定义 $f(t)$ 表示：

- 令 $n$ 表示 $t$ 的点数。
- 如果 $n=1$，那么 $f(t)=1$。
- 如果 $n\ge 2$  
  - 对于一条 $t$ 中的边 $e$，定义 $t_x(e),t_y(e)$ 分别表示删去 $e$ 后得到的两棵树。
  - $f(t)=(\sum_{e\in t}f(t_x(e))\times f(t_y(e)))/n$

现在已知一棵 $N$ 个点的树 $T$，求 $f(T)\bmod 998244353$。

## 数据范围

- $2\le N\le 5000$
- $1\le A_i,B_i\le N$

## 输入格式

第一行一个数 $N$。

接下来 $N-1$ 行，每行两个数，表示一条树边。

## 输出格式

一行一个数，表示答案。

## 样例

### 样例输入1

```
2
1 2
```

### 样例输出1

```
499122177
```

### 样例解释1

$f(T)=\dfrac{1}{2}$。
