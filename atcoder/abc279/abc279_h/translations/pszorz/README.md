### 题意

已知两个正整数 $N,M$，满足 $N\le M\le 2N$。

对于所有正整数序列 $S=(S_1,S_2,...,S_N)$ 满足 $\sum_{i=1}^N S_i=M$，定义其价值为：
$$
\prod_{k=1}^N \min(k,S_k)
$$
求所有 $S$ 价值之和对 $200003$（一个素数）取模的结果。

### 数据范围

- $1\le N\le 10^{12}$
- $N\le M\le 2N$

---

### 输入格式

一行两个数 $N,M$。

### 输出格式

一行一个数，表示答案。

---

### 样例输入1

```
3 5
```

### 样例输出1

```
14
```

### 样例解释1

合法的 $S$ 有：$S=(1,1,3),(1,2,2),(1,3,1),(2,1,2),(2,2,1),(3,1,1)$。

他们的价值分别是 $3,4,2,2,2,1$，和为 $14$。
