## 题意

有一个长 $n$ 米，宽 $m$ 米的矩形，你需要用边长为 $a$ 米的正方形去覆盖它。

要求：正方形的每条边都至少与矩形的一条边平行；允许正方形超出矩形。

至少需要多少个这样的正方形？

## 输入格式

一行三个整数 $n$ $m$ $a$。$1\leq n, m, a\leq 10^9$。

## 输出格式

一行一个整数，表示至少需要这样的正方形的个数。

## 样例

### 输入 1

```
6 6 4
```

### 输出 1

```
4
```
