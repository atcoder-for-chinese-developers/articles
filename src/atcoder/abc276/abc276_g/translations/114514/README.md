### 题意 

求有多少个长度为 $N$ 的序列 $A=(a_1,a_2,...,a_N)$ 满足以下条件，答案对 $998244353$ 取模。

- $0\le a_1\le a_2\le ... \le a_N \le M$

- 对于 $i=1,2,...,N-1$，都满足 $a_i \bmod 3\ne a_{i+1} \bmod 3$。 

### 数据范围

- $2\le N\le 10^7$

- $1\le M\le 10^7$

---

### 输入格式

一行两个数 $N,M$。

### 输出格式

一行一个数，表示答案。

---

### 样例输入1

```
3 4
```

### 样例输出1

```
8
```

### 样例解释1

以下是所有满足条件的序列：

- $(0,1,2)$
- $(0,1,3)$
- $(0,2,3)$
- $(0,2,4)$
- $(1,2,3)$
- $(1,2,4)$
- $(1,3,4)$
- $(2,3,4)$

---

### 样例输入2

```
276 10000000
```

### 样例输出2

```
909213205
```
