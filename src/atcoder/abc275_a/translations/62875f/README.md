### 题意 

给定 $N$ 个正整数 $H_1\sim H_N$，求出其中最大的数并输出它的编号（它是第几个数）。

### 数据范围

输入的所有数值都是整数。

$1 \le N\le 100$

$1 \le H_i \le 10^9$

---

### 输入格式

第一行一个数 $N$，之后一行 $N$ 个数表示  $H_1\sim H_N$ 。

```
N
H1 H2 ... Hn
```

### 输出格式

输出一个整数表示答案。

---

### 样例输入1

```
3
50 80 70
```

### 样例输出1

```
2
```

### 样例解释1

$3$ 个数中，最大的数是 $80$ ，它的编号是 $2$ 。

---

### 样例输入2

```
1
1000000000
```

### 样例输出2

```
1
```

---

### 样例输入3

```
10
22 75 26 45 72 81 47 29 97 2
```

### 样例输出3

```
9
```

### 样例解释3

$10$ 个数中，最大的数是 $97$ ，它的编号是 $9$ 。
