## 题意

小写英文字母 `a`,`b`,$\dots$,`z` 的 ASCII 编码分别是 $97,98,\dots,122$。

现给出一个在 $97$ 到 $122$ 之间的整数 $N$，请输出 ASCII 为 $N$ 的小写字母。

## 数据范围

$N$ 是一个在 $97$ 到 $122$ 之间的整数。

## 输入格式

输入是由标准输入提供的，格式如下：

```
N
```

## 输出格式

输出你的答案。

## 样例

### 样例一

```
97
```

```
a
```

### 样例二

```
122
```

```
z
```
