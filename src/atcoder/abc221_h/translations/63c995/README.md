### 题意 

已知两个正整数 $N,M$。

对于每个 $k=1,2,...,N$，解决以下问题，答案对 $998244353$ 取模。

- 求有多少个大小为 $k$ 的正整数可重集 $A$ 满足以下条件：
  - $A$ 中所有数之和为 $N$。
  - 对于每个正整数 $x$ ，$x$ 在 $A$ 中最多出现 $M$ 次。

### 数据范围

- $1\le M\le N\le 5000$

---

### 输入格式

一行两个数 $N,M$。

### 输出格式

$N$ 行，第 $i$ 行表示 $k=i$ 时的答案。

---

### 样例输入1

```
4 2
```

### 样例输出1

```
1
2
1
0
```

### 样例解释1

- $k=1$ 时，满足条件的集合是 $\{4\}$。
- $k=2$ 时，满足条件的集合是 $\{1,3\},\{2,2\}$。
- $k=3$ 时，满足条件的集合是 $\{1,1,2\}$。
- $k=4$ 时，没有满足条件的集合。

---

### 样例输入2

```
7 7
```

### 样例输出2

```
1
3
4
3
2
1
1
```
