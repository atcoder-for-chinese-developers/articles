## 题意

已知一个长度为 $N$ 的字符串序列 $s_1,s_2,...,s_N$。

现在从中选出一个子序列（可以不连续），把选出的字符串按原序列中的相对位置拼接起来，求所有可以拼接出的长度为 $K$ 的字符串中字典序最小的字符串。

## 数据范围

- $1\le N\le 2000$

- $1\le K\le 10^4$

- $1\le |s_i|\le K,\sum_{i=1}^N |s_i|\le 10^6$

- $s_i$ 中仅包含小写字母。

- 保证至少可以拼出一种长度为 $K$ 的字符串。


## 输入格式

第一行输入两个正整数 $N,K$。

接下来 $N$ 行，每行一个字符串，第 $i$ 行的字符串表示 $s_{i-1}$。

## 输出格式

一行一个字符串，表示答案。

## 样例

### 输入样例 #1

```
3 7
at
coder
codar
```

### 输出样例 #1

```
atcodar
```

### 样例解释 #1

选择 `at`，`codar` 。

### 输入样例 #2

```
3 7
coder
codar
at
```

### 输出样例 #2

```
codarat
```

### 样例解释 #2

选择 `codar`，`at`。

### 输入样例 #3

```
4 13
kyuri
namida
zzzzzzz
aaaaaa
```

### 输出样例 #3

```
namidazzzzzzz
```

### 样例解释 #3

选择 `namida `，`zzzzzzz `。
