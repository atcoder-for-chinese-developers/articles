## 题意

给定一个长度为 $N$ 的整数序列 $A$。    
求出满足 $i < j$ 且 $A_i - A_j$ 是 $200$ 整数倍的数对 $(i,j)$ 的数量。    

## 数据范围

$1\le N\le 2\times 10^5$，        
$1\le A_i\le 10^9$。

## 输入格式

一行一个整数 $N$。     
第二行 $N$ 个整数，即序列 $A$。

## 输出格式

一行一个整数表示满足条件的数对数量。

## 样例 

### 输入1 :
```
6
123 223 123 523 200 2000
```   

### 输出1 :
```
4
```  

### 输入2 :
```
5
1 2 3 4 5
``` 

### 输出 2:
```
0
```

### 输入3 :
```
8
199 100 200 400 300 500 600 200
```

### 输出3 :
```
9
```

## 样例解释

对于样例 1，满足条件的数对有 $(1,3),(1,4),(3,4),(5,6)$。
