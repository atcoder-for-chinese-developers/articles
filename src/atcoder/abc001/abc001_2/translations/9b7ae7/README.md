## 题意

设能见距离为 $x(\text{km})$。

能见度值 $V$ 计算规则如下：

1. $x<0.1$：$V$ 值为 $00$。
2. $0.1\le x\le 5$：$V=10x$。
3. $6\le x\le 30$：$V=x+50$。
4. $35\le x\le 70$：$V=\lfloor\frac{x-30}5\rfloor+80$。
5. $x>70$：$V=89$。

需要注意的是，$V$ 始终是一个带前导零的两位数。

输入一个整数 $n$，表示能见距离为 $n(\text{m})$。

## 数据范围

- $0\le n\le10^5$。
- 保证不存在非法数据。

## 样例输入 1
```
15000
```

## 样例输出 1
```
65
```

## 样例输入 2
```
75000
```

## 样例输出 2
```
89
```

## 样例输入 3
```
200
```

## 样例输出 3
```
02
```
