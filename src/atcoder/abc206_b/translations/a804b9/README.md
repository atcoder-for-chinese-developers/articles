## 题意  

第 $i$ 天，小 A 会存 $i$ 元钱，问最早在第几天小 A 的钱不小于 $N$。

## 数据范围

$1\le N\le 10^9$。      

## 输入格式

第一行，一个整数，$N$。     
          
## 输出格式

一行一个整数，表示天数。

## 样例

### 输入1 :
```
12
```

### 输出1 :
```
5
```

### 输入2 :
```
100128
```

### 输出2 :
```
447
```

## 样例解释
对于样例一，第五天小 A 的钱有 $1+2+3+4+5=15 > 12$ 元。
