## 题意  

输入字符串 $S_1,S_2,S_3,S_4$，
可以交换任意两个字符串，问能否使 
- $S_1=$ 'H'，
- $S_2=$ '2B'，
- $S_3=$ '3B'，
- $S_4=$ 'HR'。

## 数据范围
                     
$S_i$ 为 'H' '2B' '3B' 'HR' 中的一个。      

## 输入格式

四行，$S_{1 \cdots 4}$。        
          
## 输出格式

一行 'Yes' 或 'No' 表示能否满足条件。     

## 样例

### 输入1 :
```
3B
HR
2B
H
```

### 输出1 :
```
Yes
```

### 输入2 :
```
2B
3B
HR
3B
```

### 输出2 :
```
No
```
