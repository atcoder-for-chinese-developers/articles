## 题意

给定 $L,R$，求合法数对 $(x,y)$ 的个数，

一个数对是合法的，当且仅当：

- $L\le x,y\le R$
- $\gcd(x,y)\neq 1,\frac{x}{\gcd(x,y)} \neq 1. \frac{y}{\gcd(x,y)} \neq 1$

## 数据范围

$1\le L\le R\le 10^6$。

输入的都是整数。

## 输入格式

一行两个整数 $L,R$。

## 输出格式

一行一个整数，合法数对的数量。

## 样例

### 样例输入 1

```
3 7
```

### 样例输出 1

```
2
```

### 样例输入 2

```
4 10
```

### 样例输出 2

```
12
```

### 样例输入 3

```
1 1000000
```

### 样例输出 3

```
392047955148
```

## 样例解释

对于样例 1，合法的数对有 $(4,6),(6,4)$。
