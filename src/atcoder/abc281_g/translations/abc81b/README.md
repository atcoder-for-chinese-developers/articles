## 题意

给出两个正整数 $N,M$。

设 $dis_{i,j}$ 为从 $i$ 到 $j$ 所经过的最少边数。

找出满足以下条件的、不同的、有 $N$ 个点的简单无向连通图的数量：

- $\forall u \in [2,N-1],dis_{1,u} < dis_{1,N}$

答案对 $M$ 取模。

## 数据范围 

- $3 \leq N \leq 500$

- $10^8 \leq M \leq 10^9$

- 所有输入均为整数。

## 输入格式

第一行输入两个正整数 $N,M$。

## 输出格式

第一行输出答案对 $M$ 取模后的结果。

## 样例

### 输入样例 #1

```
4 1000000000
```

### 输出样例 #1

```
8
```
