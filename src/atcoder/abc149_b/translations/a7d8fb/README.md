### 题意 
甲和乙分别有 $A,B$ 个饼干，进行 $K$ 次操作，第一次操作如果甲有饼干吃掉甲的一块饼干，第二次操作如果乙有饼干吃掉乙的一块饼干，第三次操作如果甲有饼干吃掉甲的一块饼干……问你最后甲和乙分别有几块饼干。
### 数据范围
$A,B,K\le10^{12}$

---
### 输入格式
一行 $3$ 个整数 $A,B,K$
### 输出格式
一行 $2$ 个整数表示甲和乙最后的饼干数量。

---
### 样例输入1
```
2 3 3
```
### 样例输出1
```
0 2
```
### 样例解释1
吃甲吃乙再吃甲，甲 $0$，乙 $2$

---
### 样例输入2
```
500000000000 500000000000 1000000000000
```
### 样例输出2
```
0 0
```
