## 题意  

给定一个只含 `01689` 的字符串 $S$，输出 $S$ 翻转 $180$ 度后的字符串。    
**注意：`018` 翻转后不变，`6` 翻转后是 `9`，`9` 翻转后是 `6`。** 

## 数据范围

$1\le \mid S\mid \le 100$。

## 输入格式

一行，字符串 $S$。
          
## 输出格式

一行，翻转后的字符串。

## 样例

### 输入1 :
```
0601889
```

### 输出1 :
```
6881090
```

### 输入2 :
```
86910
```

### 输出2 :
```
01698
```

### 输入3 :
```
01010
```

### 输出3 :
```
01010
```
