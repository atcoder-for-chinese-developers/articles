## 题意
给出一个大于等于 $100$ 的正整数 $N$，请按照十位，个位的顺序输出 $N$ 的最后两位。

## 数据范围

- $100 \le N \le 999$
- $N$ 是整数。

## 输入格式

输入是由标准输入提供的，格式如下：

```
N
```

## 输出格式

输出一行答案。

## 样例

### 样例输入1

```
254
```

### 样例输出1

```
54
```

### 样例解释1

$254$ 的最后两位是 $54$，所以应该输出 $54$。

### 样例输入2

```
101
```

### 样例输出2

```
01
```

### 样例解释2

$101$ 的最后两位是 $01$，所以应该输出 $01$。
