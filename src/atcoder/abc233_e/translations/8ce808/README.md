### 题意
求 $\sum_{k=0}^{10^{100}}\lfloor\dfrac{x}{10^k}\rfloor$
### 数据范围
$1\le x < 10^{500000}$

---
### 输入格式
一行 $1$ 个整数 $x$。  
### 输出格式
一行一个整数表示值。
要求采取整型输出，不允许采用科学计数法或出现前导零。

---
### 样例输入1
```
1225
```
### 样例输出1
```
1360
```
### 样例解释1
$1225+122+12+1+0+0+...+0=1360$

---

```
99999
```
### 样例输出2
```
111105
```
