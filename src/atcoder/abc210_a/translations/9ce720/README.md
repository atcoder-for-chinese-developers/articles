## 题意  

你要购买 $N$ 棵卷心菜。        
每棵卷心菜 $X$ 元，    
但是如果你购买了超过 $A$ 棵卷心菜，    
那么其中 $A$ 棵卷心菜每棵 $X$ 元，剩下的卷心菜每棵 $Y$ 元。      
问你要付款多少元。                     

## 数据范围
                     
$50\le N,A\le 300$，      
$1\le Y < X\le 100$，                                     
所有输入的数都是整数。     

## 输入格式

第一行四个整数 $N,A,X,Y$。         
          
## 输出格式

一个数，表示你花费的钱数。       

## 样例

### 输入1 :
```
5 3 20 15
```

### 输出1 :
```
90
```

### 输入2 :
```
10 10 100 1
```

### 输出2 :
```
1000
```

## 样例解释

对于样例 1，$20+20+20+15+15=90$。
