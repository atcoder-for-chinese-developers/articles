## 题意

有一个含有 $N$ 个数由 $1,2$ 构成的序列 $A$，你不知道这个序列的具体内容。

但是你知道 $M$ 条限制，每条限制形如 $A_{X_i}+A_{Y_i}+Z_i$ 为偶数。

请你确定你最少需要确定多少个序列中的数才能进而确定整个序列。

## 数据范围

- $2\le N\le10^5$。
- $1\le M\le10^5$。
- $1\le X_i< Y_i\le N$。
- $1\le Z_i\le 100$。

## 输入格式

第一行两个整数 $N,M$。

接下来 $M$ 行，第 $i$ 行三个整数 $X_i,Y_i,Z_i$ 表示一条限制。

## 输出格式

一行一个整数，答案。

## 样例输入 1

```
3 1
1 2 1
```

## 样例输出 1

```
2
```

## 样例解释 1

确定 $A_1,A_3$ 或 $A_2,A_3$ 即可。

## 样例输入 2

```
6 5
1 2 1
2 3 2
1 3 3
4 5 4
5 6 5
```

## 样例输出 2

```
2
```

## 样例输入 3

```
100000 1
1 100000 100
```

## 样例输出 3

```
99999
```
