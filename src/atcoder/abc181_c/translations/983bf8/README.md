## 题意
平面直角坐标系上有 $N$ 个点，问是否有三个点共线。
## 数据范围
$3\le 10^2$

$|x_i|,|y_i|\le 10^3$

没有坐标相同的点。
## 输入格式
第一行读入一个 $N$。

接下来 $N$ 行，每行两个数 $x_i,y_i$ 表示这个点。
## 输出格式
有三点共线输出 `Yes` 否则输出 `No`。
### 样例输入1
```
4
0 1
0 2
0 3
1 1
```
### 样例输出1
```
Yes
```
### 样例解释1
$x=0$
### 样例输入2
```
14
5 5
0 1
2 5
8 0
2 1
0 0
3 6
8 6
5 9
7 9
3 4
9 2
9 8
7 2
```
### 样例输出2
```
No
```
### 样例输入3
```
9
8 2
2 3
1 3
3 7
1 0
8 8
5 6
9 7
0 1
```
### 样例输出3
```
Yes
```
